Online Finance Management

Online tool managing your daily expenses

Online Finance Management is an application which would manage the monthly financial activities of an individual or a household. The aim was to digitalize the process instead of maintaining these expenses in form of a notebook. This would mean that the user would just need to enter their daily expenses and update them, and the rest of the tracking would be done by the app. The user can maintain their bills as well as their bank accounts from this website, where majority of the work would be done by the system Used Java with JDBC for handling of the MySQL database and for the backend of the system, Apache TOMCAT as the server and HTML, CSS3 and Java Server Pages for designing the front end of the system.



Face Recognition

Recognizes faces from the database.

This project report provides a general overview of face recognition. Face detection is the process of finding the face in the entire image and detecting the various facial features. In face recognition the system recognises the face through various parameters like Eigen faces, Fisher faces, binary values etc. The faces in the training database are compared to the one in the test database to find whether the face exists in the database or not. A few face detection techniques and recognition algorithms have been used. Their flaws and benefits have been compared amongst each other to find out the most favorable based on certain conditions. In the end we conclude by proposing a system, which we believe, provides better results. MATLAB is used for the process of Face dectection and Recognition



Image Denoising

Denoising Images using different algorithms

Images consist of noise, ‘Image Noise’ is the extra spurious and unwanted information in the image, which obstructs from giving the actual image and distorts the image. Image noise is random variation of brightness or colour information in images and usually an aspect of electronic noise. Noise gets introduced into the data via any electrical system used for storage, transmission, and/or processing. This project focused on the removal of unwanted noise from the image. The six main types of noise were studied and then the three types of filters namely spatial, wavelet and frequency filters were applied and compared to see which was more efficient in removing noise. The comparison done here was based on the results obtained by coding in MATLAB
